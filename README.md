# Joint

#### 兼容低版本ie
### joint.js -跳转拼接
### joint2.js -不跳转拼接



#### 使用方法：
joint.js:
```
joint.add('name','param')       //添加参数
joint.remove('name')            //移除参数
joint.replace('name','param')   //替换参数
joint.get('name')               //获得参数值
joint.clear()                   //清空参数
```
joint2.js:
```
joint2.add('name','param')       //添加参数
joint2.remove('name')            //移除参数
joint2.replace('name','param')   //替换参数
joint2.get('name')               //获得参数值
joint2.clear()                   //清空参数
```
